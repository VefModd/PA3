##Setup
1. Run the **setup.sh** script or equivalent commands to the ones below
```
	npm install -d
	bower install -d
	grunt
```
2. Run **run_client.sh** script or equivalent commands to the ones below
```
	python -m SimpleHTTPServer 8088
```
3. Go to localhost:8088 and have fun.
